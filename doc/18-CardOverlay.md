# 18 - CardOverlay — 卡牌选择界面

**文件路径**: `src/rendering/CardOverlay.ts` (136 行)

## 概述

关卡通过后显示的卡牌选择 UI，覆盖整个画面。

## 类：CardOverlay

### 布局

```
┌─────────────────────────────┐
│      STAGE X CLEAR!         │  ← 标题（黄色大字）
│      选择一项强化             │  ← 副标题
│                             │
│  ┌──────┐ ┌──────┐ ┌──────┐ │
│  │ 名称  │ │ 名称  │ │ 名称  │  ← 彩色标题栏
│  │      │ │      │ │      │ │
│  │ 描述  │ │ 描述  │ │ 描述  │  ← 卡牌说明
│  │      │ │      │ │      │ │
│  │点击选择│ │点击选择│ │点击选择│  ← 操作提示
│  └──────┘ └──────┘ └──────┘ │
└─────────────────────────────┘
```

- 卡牌尺寸：130×210 像素
- 间距：16px
- 3 张卡牌水平居中排列

### 方法

| 方法 | 签名 | 说明 |
|------|------|------|
| `show()` | `(cards, clearedStage, onSelect) => void` | 显示卡牌选择 |
| `hide()` | `() => void` | 隐藏并清理 |

### 交互

- **悬停**：卡牌边框加粗变亮，提示文字变黄
- **点击**：隐藏覆盖层，回调 `onSelect(card)`
- 使用 Phaser Zone 实现交互区域

### 卡牌绘制

每张卡牌包含：
- 圆角矩形背景（深蓝色）
- 彩色标题栏（卡牌主题色）
- 名称（白色粗体）
- 描述文字（浅蓝色）
- 操作提示

### show() 流程

1. 清空容器
2. 绘制全屏半透明黑色遮罩
3. 显示标题和副标题
4. 为每张卡牌创建交互式 UI
5. 显示容器（`setVisible(true)`）

### hide() 流程

1. 隐藏容器
2. 移除所有子元素（`removeAll(true)` 递归销毁）

### 依赖

- `Phaser.Scene`
- `PlayerUpgrades` — `UpgradeCard` 类型
- `constants.ts` — 画布尺寸
