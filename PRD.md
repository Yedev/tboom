# T-Boom 产品需求文档 (PRD)

## 1. 产品概述

T-Boom 是一款融合俄罗斯方块与平台跳跃的创新混合游戏。玩家同时操控两个维度：下落的俄罗斯方块和棋盘内的角色角色。角色可以在已锁定的方块上奔跑跳跃，使用炸弹破坏方块，但必须躲避方块碾压和爆炸伤害。

**核心卖点**：俄罗斯方块 + 平台跳跃 + 炸弹破坏 = 三重策略博弈

## 2. 目标用户

- 休闲手游玩家（竖屏操作）
- 喜欢创新玩法的俄罗斯方块爱好者
- 喜欢像素风格的独立游戏玩家

## 3. 平台与技术

| 项目 | 规格 |

## 4. 游戏机制

### 4.1 俄罗斯方块系统

- **棋盘**：10列 x 20行，标准尺寸
- **方块**：7种标准方块（I/J/L/O/S/T/Z），4个旋转状态
- **旋转**：SRS（Super Rotation System）标准踢墙数据
- **跟随模式**：新方块跟随角色X位置自动移动，方块同时受重力下落
- **确认放置**：按 W 或触摸「放置」按钮，方块停止跟随，从当前位置继续下落
- **硬降**：按 S 或触摸「速降」按钮，方块瞬间落到底部
- **锁定延迟**：500ms，最多15次移动延长锁定
- **消行**：满行消除，400ms闪烁动画
- **计分**：1行=100, 2行=300, 3行=500, 4行=800 × 当前等级
- **升级**：每10行升1级，下落速度加快（800ms起步，每级-40ms，最低50ms）

### 4.2 角色系统

- **尺寸**：20 x 24 像素
- **生命值**：5 HP，用心形显示
- **移动**：左右移动 160 px/s
- **跳跃**：初速度 -420 px/s（向上），重力 1200 px/s²
- **碰撞**：AABB 碰撞检测，分轴解算（先X后Y）
- **碾压伤害**：方块锁定时覆盖角色位置 → 扣2 HP + 推到最近安全位置
- **无敌帧**：受伤后1500ms无敌，闪烁效果
- **死亡条件**：HP=0 或 角色掉出棋盘底部
- **动画**：待机（呼吸压缩）、行走（腿部交替）、跳跃（收腿举臂）、下落（展开）、受伤（红色+X眼）

### 4.3 炸弹系统

- **获取**：每锁定一个方块获得 +1 炸弹
- **放置**：角色脚下生成，受重力影响（800 px/s²）
- **引信**：2000ms，在 1200/800/400ms 时发出滴答声
- **爆炸半径**：
  - 破坏半径：1格（3x3 范围内方块被摧毁）
  - 伤害半径：2格（角色扣2 HP）
- **提前引爆**：下落的俄罗斯方块碰到炸弹立即引爆
- **视觉**：爆炸 400ms 橙黄色闪烁

### 4.4 方块-角色联动

- 方块下落时检测与炸弹碰撞 → 引爆
- 方块锁定后检测角色碾压 → 伤害 + 推开
- 消行时角色脚下方块消失 → 角色自然下落
- 角色站在方块上 → 方块作为平台

## 5. 操作方式

### 5.1 键盘（桌面）

| 按键 | 功能 |
|------|------|
| A / D | 角色左右移动 |
| Space | 角色跳跃 |
| E | 放置炸弹 |
| Q | 旋转方块 |
| W | 确认放置（方块停止跟随） |
| S | 硬降 |
| P | 暂停 |
| R | 重新开始（游戏结束时） |

### 5.2 触屏（移动端）

**左侧 — 虚拟摇杆**：
- 全方向拖动控制角色移动
- 向上推超过50%触发跳跃

**右侧 — 圆形按钮**（中文标签）：

| 按钮 | 功能 |
|------|------|
| 旋转 | 旋转方块 |
| 放置 | 确认方块位置 |
| 炸弹 | 放置炸弹 |
| 速降 | 硬降 |
| 暂停 | 暂停游戏 |

- 支持多点触控（最多4个触点）
- 按钮使用 consume 模式（每次按压只触发一次）

## 8. 游戏流程

```
开始游戏
  ↓
生成新方块（跟随角色X位置）
  ↓
角色在棋盘内移动/跳跃/放炸弹
方块同时受重力下落
  ↓
玩家确认放置(W) 或 硬降(S)
  ↓
方块锁定 → 写入棋盘
  ├→ 检测角色碾压 → 扣血/推开
  ├→ 炸弹+1
  └→ 检测消行 → 计分/升级
  ↓
角色存活？ ──否──→ 游戏结束 → 按R重开
  │是
  ↓
返回"生成新方块"
```
